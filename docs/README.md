## Overview

Slow-Check is a lightweight Node.js utility that wraps command execution with visual progress indicators and timing information. It's designed to improve the user experience when running long-running commands by providing real-time feedback about execution status.

The tool is built with simplicity and reliability in mind, using only Node.js standard libraries and following Unix philosophy principles.
